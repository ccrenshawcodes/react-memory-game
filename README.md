# Studio Ghibli-themed memory game for The Odin Project
Test your memory and your love of Studio Ghibli films with this Ghibli-themed memory game!

## Built With:
- Vite + React
- [Canvas Confetti] (https://github.com/catdad/canvas-confetti)
